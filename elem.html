Add a table

  1. Now we will add a table to the footer. Add a empty table into the footer-tag
    2. <table> <tr> <th></th> <th></th> <th></th> </tr> <tr> <td></td> <td></td> <td></td> </tr> </table>
  2. Table rows consist tr and a td of table header or table data. Add text/image/links to the table header th and table data td elements.
    3. <tr> <th>The course</th> </tr> <tr> <td>Introduction to Digital Services</td> </tr>

  3. Style your table with borders and align text to the left of the table cell. Here the table is in the footer that has a blue background, so the text color is styled to be white.

    As we might have tables in other pages on the site, we define class table-class-footer for the table in the footer.
    4. .table-class td { color:#FFFFFF; text-align:left; padding: 8px; border: 2px solid lightblue; } .table-class th{ color:#FFFFFF; text-align:left; padding: 8px; border: 2px solid lightblue; }

  4. Links are blue as default, so we want to change the link color in the class table-style-footer.

    5. .table-class a { color: #FFFFFF; }
  5. To make the footer responsive with the table and the Navi we need to make it a flex container. We do this by adding .flex-container-footer class to it.
    6. <footer class="flex-container-footer">
  6. In this flex container we want the items felx direction to be horizontal, so that they are always in a colunm. It also defines the backgoound color of the footer.

  7. Add a new class to your style file.

    8. .flex-container-footer { display: flex; flex-direction: column; background-color: rgb(13, 92, 172); }
    Now we still need to define the table and Navi as flex items of the outer container by adding flex-item class to them.
    <table class="table-class flex-item"> <nav class="flex-container felx-item">

Some images to the site

  1. Add a picture to your page with img tag and define the style in CSS. To fit the image to the page and make it responsive some in-line styling may be needed.
    2. <img src="ai-generated-8277268_1920.jpg" alt="Summer and flowers" width=80%>

  2. If you want to add a caption to your image, the figure tag might be more useful

    3. <figure> <img src="ai-generated-8188708_640.jpg" alt="A digital hart" width="100%"> <figcaption>A digital hart</figcaption> </figure>

  3. You can add some style to the figure

    4. figure { border: 1px #cccccc solid; padding: 4px; margin: auto; } figcaption { background-color: black; color: white; font-style: italic; padding: 2px; text-align: center; }
