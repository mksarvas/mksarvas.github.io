Esimerkki lomakkeesta

Tälläs sivulla on esimerkki lomakkeesta joka tyyli on määritelty CSS:ssä. Lomaketta ei kuitenkaan voi toteuttaa pelkästään HTML:llä, koska sen täyttämisen jälkeen tarvitaan jokin toiminta joka käsittelee data mikä lomakkeelle syötetään. Yleensä tähän käytetään JavaScriptiä.

Myös jokin paikka minne lomakkeella saadut tieto tallennetaan olisi hyvä olla, vaikka tietokanta. Tähänä taas tarvimme jonkin rajapinnan minkä kautta keskustella teitokannan kanssa.

Lomakkeen tyyli

.lomake { background-color: lightgray; color: blue; padding: 20px; border-radius: 10%; } /* NAVIVALIKKO VASEN*/ .navi{ float: left; width: 30%; } .navi > div { margin: 10px; }
