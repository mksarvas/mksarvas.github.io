Here is step-by-step guide bulding a web site on the course Digi Intro

Create pages with the basic elements

  1. Createa HTML file named index.html
  2. Add head and write <html>,<head>,<body>tags into it.

    TIP: In Vs Code with writing html:5 and pressing enter, you get the beginning tags for your HTML document.

    3. <!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Document</title> </head> <body> </body> </html>
  3. Add a <title> to your page. Notice you have the tag already if you did as told in the previous TIP
  4. Write meta keywords and meta description to your page, so that they match the content of your site. <meta name="keywords" content="Sarvas, It Teacher, Inspiring"> <meta name="description" content="Inspiring teacher, Customer solution specialist, Self-employed consult - Programming, Web solutions, Project Management">
  5. Create the 2 other pages for your site. For example one.html and two.html
  6. Copy the <head> tag and it's content from index.html to the new pages. Change the titles to match the content of the pages.
  7. Link all your 3 page together by writing the following code into the begginig of each page into a semantic <nav> tag
    8. <nav><a href="index.html">INDEX</a> <a href="one.html">ONE</a> <a href="two.html">TWO</a> </nav>
  8. Write the right filenames, - your files, where needed. We will work making the navigation look nicer and responsible later.

    9. CSS

  9. Create a style.css file
  10. Link it to all your 3 pages by adding the following line to the head of the file.
    11. <link rel="stylesheet" href="styles.css">
  11. Add style to your styles.css file. So at least background, headings, text. Below is one example of what it could look like.
  12. Add comments to your CSS file, so you can easily find where what style is defined
    13. /* Basic style - to all elements on the page */ /*Text style*/ .main{ margin-left: 32%; margin-right: 10%; } .main body { font-family: "Kode Mono", monospace; font-optical-sizing: auto; font-weight: 400; font-style: normal; margin-left: 25px; background-color: aqua; } /*Heading styles */ .main h1 { font-family: 'Pacifico'; text-shadow: 1px 1px; color: rgb(122, 121, 120); }
  13. Next we will start building a responsive good looking navigation. Go to next page to continue
