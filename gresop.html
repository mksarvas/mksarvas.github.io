Make your Navi look nice and responsive

The look of your Navi - CSS

  1. Start with making your navi-link into buttons.

    NOTICE HTML tag button is useful when you have JavaScrpt to use. This code is not good for accessibility. You can read more about it here.

    Instead we make the link look like a button with CSS. Define a nice looking CSS fro example clicking the button below or define one yourself.

    Button CSS's

    On the page click the button which style you want. That will copy the CSS. Then paste it to your style-file.

  2. Now make a link and style it with the button style you defined in the previous step. Here as an example the Index link in your Navi.
    3. <a href="index.html" class="button-47">Home</a>
  3. Add the same style to the other links in your Navi

Resposive Navi - Flexbox

  1. a flexcontainer class for your Navi Add the style class to your style file.
    2. .flex-container { display: flex; flex-wrap: wrap; text-align: center; margin-bottom: 30px; } Add the class to your Navi. <nav class="flex-container">
  2. Next add a flex-itme class and style your links in the Navi with it .felx-item { flex: 1; margin: 5px; }

Different styles for different media types/devices.

  1. Using media queries are a popular technique for delivering a tailored style sheet (responsive web design) to different devices.

    Here we make the background color change when the viewpoint changes. It is an easy way to see how your code and rule work. Then you can use it for other needed modifications in the style.

    NOTICE you might want to remove the changing background color from your final work.

  2. Add the @media rules to your style file.

    <
    3. @media only screen and (min-width: 421px) and (max-width: 1024px){ body{ background-color: lightyellow} } @media only screen and (min-width:1024px){ body { background-color:rgba(231, 99, 71, 0.1); }
  3. Now for practice and better look, let's change the border color of the table to match the background changes with the viewpoint. Add a new lines to each media rule.
    4. @media only screen and (min-width: 421px) and (max-width: 1024px){ body{ background-color: lightyellow;} .table-class th{ border: 2px solid lightyellow;} .table-class td{ border: 2px solid lightyellow;} } @media only screen and (min-width:1024px){ body{ background-color:lightgrey;} .table-class th{ border: 2px solid lightgray;} .table-class td{ border: 2px solid lightgrey;} }
