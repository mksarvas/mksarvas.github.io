Aloitus sivu, eli index.html

Nämä sivut on tehty esimerkiksi Haaga-Helian kurssille Johdatus Digitaalisiin Palveluihi, jossa opetellaan palvelumuotoilun ja suunnittelun lisäksi kirjoittamaan HTML-koodia sekä siihen liitettävää css:ää. Nämä sivut keskittyvät HTML ja CSS esimerkkeihin. Tämä sivu on toteutettu jakamalla sivu kaheen semaattiseen osioon aside, joka sisältää responsiivisen navigaation - ja article joka on itse leipäteksti ja muu joka sivuilla muuttuva sisältö.

Navigaatio

Navigaation on toteutettu navi-luokan tyylillä.

.navi{ float: left; width: 30%; } .navi > div { margin: 10px; } Linkkeinä eri sivuille on käytetty muotoiltuja painikkeita. Muotoilu on haettu sivulta Buttons sivulta löytyvän “Heinoja nappeja” linkin takaa löytyvästä sivusta. Navigaatiopainikkeiden teksti pienenee kun näyttö pienenee, jotta ne pysyvät painikkeiden sisällä. <aside class="navi"> <div> <a href="index.html"><button role="link" class="button-72">Home</button></a> </div> <div> <a href="kuva ja responsiivisuus.html"><button role="link" class="button-72">Image</button></a> </div> <div> <button onclick="lomake.html" role="link" class="button-72"> Form</button></a> </div> <div> <a href="button.html"><button role="link" class="button-72">Button</button></a> </div> <div> <a href="animaatio.html"><button role="link" class="button-72">Anim</button></a> </div> </aside>

Leipäteksti

Teksti on kirjoitettu article-tagiin. HTML-koodi on saatu näkymään xmp-tägin avulla, Tyyli on määritelty main-luokalla.

Navigaation aside-osiollo on varttay 32% sivun leveydestä.

.main{ margin-left: 32%; margin-right: 10%; } .main body { font-family: Garamond, serif; margin-left: 25px; background-color: aqua; } .main h1 { font-family: 'Pacifico'; text-shadow: 1px 1px; color: rgb(122, 121, 120); } .main img { max-width:300px; width:100% }
