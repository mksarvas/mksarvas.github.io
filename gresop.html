Make your Navi look nice and responsive

    The look of your Navi - CSS

  1. Start with making your navi-link into buttons.

    NOTICE HTML tag button is useful when you have JavaScrpt to use. This code is not good for accessibility. You can read more about it here.

    Instead we make the link look like a button with CSS. Define a nice looking CSS fro example clicking the button below or define one yourself.

    Button CSS's

    On the page click the button which style you want. That will copy the CSS. Then paste it to your style-file.

  2. Now make a link and style it with the button style you defined in the previous step. Here as an example the Index link in your Navi. <a href="index.html" class="button-47">Home</a>
  3. Add the same style to the other links in your Navi

    4. Resposive Navi - Felxbox

  4. a flexcontainer class for your Navi Add the style class to your style file. .flex-container { display: flex; flex-wrap: wrap; text-align: center; margin-bottom: 30px; } Add the class to your Navi. <nav class="flex-container">
  5. Next add a flex-itme class and style your links in the Navi with it
    6. .felx-item { flex: 1; margin-right: 5px; }
