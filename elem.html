    Add a table

  1. Now we will add a table to the footer. Add a empty table into the footer-tag <table> <tr> <th></th> <th></th> <th></th> </tr> <tr> <td></td> <td></td> <td></td> </tr> </table>
  2. Table rows consist < tr> of table hedaer or table data. Add text/image/links to the table header < th> and table data < td> elements. <tr> <th>The course</th> </tr> <tr> <td>Introduction to Digital Services</td>
  3. Style your table with borders and align text to the left of the table cell. Here the table is in the footer that has a blue background, so the text color is styled to white.

    As we might have tables in other pages on the site, we define class table-class-footr for the table in the footer. .table-class td { color:#FFFFFF; text-align:left; padding: 8px; border: 2px solid lightblue; } .table-class th{ color:#FFFFFF; text-align:left; padding: 8px; border: 2px solid lightblue; } Links are blue as default, so we want to chnge the link color in the class table-style-footer. .table-class a { color: #FFFFFF; }
  4. To make the footer responsive with the table and the Navi we need to make it a flex container. We do this by adding .flex-container-footer class to it. <footer class="flex-container-footer"> In this flex container we want the items felx direction to be horizontal, so that they are always in a colunm. It also defines the backgourn color of the footer. Add a new class to your style file. .flex-container-footer { display: flex; flex-direction: column; background-color: rgb(13, 92, 172); }
  5. Now we still need to define the table and Navi as flex items of the outer container by adding flex-item class to them. <table class="table-class flex-item"> <nav class="flex-container felx-item">
