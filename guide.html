Here is a step-by-step guide for building a website in the Digi Intro course

Create pages with the basic elements

  1. Createa an HTML file named index.html

  2. Add head and write html, head ,body tags into it.

    TIP: TIP: In VS Code, by typing html:5 and pressing enter, you get the starting tags for your HTML document-

    3. <!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Document</title> </head> </body> </html>

  3. Add a ;title; to your page. Notice you have the tag already if you did as told in the previous TIP

  4. Write meta keywords and meta description to your page, so that they match the content of your site. <meta name="keywords" content="Sarvas, It Teacher, Inspiring"> <meta name="description" content="Inspiring teacher, Customer solution specialist, Self-employed consult - Programming, Web solutions, Project Management">

  5. Create the 2 other pages for your site. For example one.html and two.html

  6. Copy the head tag and it's content from index.html to the new pages. Change the titles to match the content of the pages.

  7. Link all your 3 page together by writing the following code into the begginig of each page into a semantic nav tag
    8. <nav> <a href="index.html">INDEX</a> <a href="gresop.html">ONE</a> <a href="elem.html">Elements</a> </nav>
  8. Write the right filenames, - your files, where needed. We will work making the navigation look nicer and responsible later.

CSS

  1. Create a style.css file

  2. Link it to all your 3 pages by adding the following line to the head of the file.
    3. <link rel="stylesheet" href="styles.css">

  3. Add style to your styles.css file. So at least background, headings, text. Below is one example of what it could look like.

  4. Add comments to your CSS file, so you can easily find where what style is defined
    5. /* Basic style - to all elements on the page */ body { background-color:lightcyan; font-style: normal; } h1 { font-family: 'Pacifico'; text-shadow: 1px 1px white; color: rgb(122, 121, 120); } h2 { font-family:'Franklin Gothic Medium', 'Arial Narrow', Arial, sans-serif; text-decoration:underline; color: blue; }
